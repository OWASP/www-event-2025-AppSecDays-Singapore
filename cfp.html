Call for Papers

Share your expertise with the cybersecurity community at OWASP AppSec Days Singapore 2025.

Submit Your Proposal

Important Dates

  • CFP Opens 1 April 2025
  • CFP Closes 31 May 2025
  • Speaker Notification 30 June 2025

Topics of Interest

We welcome proposals on a wide range of application security topics, including but not limited to:

  • Web Application Security
  • API Security
  • Cloud Security
  • Mobile Security
  • DevSecOps
  • Container Security
  • Secure Coding Practices
  • Security Testing
  • Threat Modeling
  • Zero Trust Architecture

Session Types

  • Talks (45 minutes): Presentations on research, case studies, or practical experiences.
  • Workshops (Full day): Hands-on sessions focused on specific tools or techniques.

Selection Criteria

Proposals will be evaluated based on:

  • Relevance to application security
  • Technical depth and accuracy
  • Originality and innovation
  • Practical applicability
  • Speaker experience and expertise
