Onsite AppSec CTF

September 12, 2025

Jeopardy Style CTF with 6 Labs

We're preparing an exciting onsite jeopardy-style CTF experience focused on Application Security for AppSec Days Singapore 2025 attendees. The competition will feature 6 comprehensive labs covering various aspects of application security.

The onsite CTF will feature hands-on challenges related to:
  • Web application vulnerability discovery and exploitation
  • Secure coding practices
  • API security testing
  • Mobile application security
  • Authentication and authorization bypasses
  • Server-side vulnerabilities
